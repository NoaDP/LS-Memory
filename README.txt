READMEPer tal de poder executar el nostre programa, s’ha de seguir uns passos molt clars que explicarem acontinuació:1. Obrir el fitxer .sql que conté la base de dades del nostre projecte amb l’aplicació MYSQL Workbench, i executar la base de dades.2. Obrir els projectes  “Memory_Server” i “Memory_Client” amb el programa Eclipse. 3. Primerament cal executar el programa “Memory_Server” accedint primer al package “model” i executant la classe “Main”. A continuació apareixerà una finestra gràfica en la que es mostraràn uns camps a omplir. Aquests camps serveixen per establir una hora d’inici del torneig i la durada d’aquest.4. Un cop inicialitzat el servidor es podrà executar el programa client, executant la classe “Main” situada al package “model” del projecte.Funcionament:El projecte està compost per dues parts: servidor i client.  El programa servidor ens permet gestionar la competició i els usuaris que s’inscriuen. Està compost per un JFrame que conté 4 opcions:1.	Gestió de la competició: Permet establir la hora d’inici del torneig i la durada, comprovar quant temps queda fins que finalitzi el torneig, comprovar quantes partides  s’han realitzat fins el moment i veure quin jugador està a la primera posició del rànquing.2.	Gestionar Usuaris: Permet registrar un nou usuari a la base de dades de la competició, visualitzar els usuaris enregistrats fins el moment i el seu estat i permet bloquejar o desbloquejar un usuari clickant amb el botó esquerra del ratolí sobre el seu nom.3.	 Mostrar Classificació: Mostra el rànquing de la competició. En aquest apartat es mostra el nom dels usuaris registrats i la seva puntuació acumulada fins el moment en ordre descendent  de puntuació.4. Mostrar Gràfic: Aquesta opció ens permet mostrar el gràfic de les puntuacions obtingudes per partida per usuari per les modalitats de joc del mode Time Trial. El programa client és l’encarregat de permetre als usuaris jugar a les diferents modalitats de memory i participar a la competició. El menú principal del programa client està compost per 8 opcions:1. Register: Aquesta opció permet als usuaris crear un nou compte i emmagatzemar la informació a la base de dades del sistema.2. LogIn: Aquesta opció permet als usuaris començar la seva sessió sempre i quan s’haguin registrat en la competició prèviament.3. Enter as Guest: Aquesta opció permet als usuaris accedir als jocs sense la necessitat de registrar-se, però a comparació amb els altres, si l’usuari inicia sessió com a guest, no participarà al torneig, sinó que jugarà en mode entrenament.4. Time Trial: Aquesta opció permet al usuari jugar al memory en mode “Classic” (el memory de sempre) o “Focus”(es mostren les cartes al començament i l’usuari les ha de encertar) . També permet seleccionar la dificultat en la que es vol jugar a aquestes dues modalitats.5. Vs. Machine:	Com el seu nom indica, aquesta opció permet jugar contra la màquina. La opció també permet seleccionar la dificultat de joc.6. Ranking: Aquesta opció fa una consulta al servidor del rànquing de la competició i el mostra per pantalla.7. Game Information: Aquesta opció mostra una finestra amb la informació de la assignació de punts per modalitat de joc.8. Log Out.: Per últim, aquesta opció permet desconnectar-se del programa i tancar sessió.
